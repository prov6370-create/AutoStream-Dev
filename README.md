# 🎮 Steam Price Monitor & Telegram Bot

**Професійна система автоматичного моніторингу цін у Steam із миттєвими сповіщеннями.**

Цей проект автоматизує процес відстеження знижок на ігри. Скрипт самостійно імітує дії користувача, проходить перевірку віку, аналізує вартість гри та сповіщає користувача в Telegram, якщо ціна змінилася.

---

## 🛠 Технологічний стек
* **Python 3.x** — основа проекту.
* **Playwright** — для парсингу динамічного контенту та обходу вікових обмежень.
* **Aiogram 3.x** — сучасний фреймворк для роботи з Telegram Bot API.
* **Pandas** — для зберігання та порівняння історії цін у CSV.
* **Regex** — для точної обробки фінансових даних.

---

## ✨ Основні можливості
1. **Age-Gate Bypass:** Автоматичний обхід вікових фільтрів Steam (вибір року народження 2000+).
2. **Smart Comparison:** Скрипт порівнює поточну ціну з попередньою та реагує лише на реальні зміни.
3. **Telegram Alerts:** Детальні сповіщення, що включають:
    * Статус ціни (📉 впала / 📈 виросла).
    * Назву гри.
    * Стару та нову ціну.
    * Різницю у валюті (грн).
    * Пряме посилання на сторінку гри.
4. **Anti-Detection:** Використання реальних заголовків браузера для стабільної роботи.

---

## 📂 Структура проекту
* `main.py` — логіка парсингу, обробка даних та порівняння цін.
* `TG_Boot.py` — модуль Telegram-бота для відправки сповіщень.
* `steam_prices.csv` — локальна база даних (автоматично створюється скриптом).
